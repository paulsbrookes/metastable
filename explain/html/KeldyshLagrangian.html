1. Starting Point: The Lindblad Master Equation

The Lindblad master equation for the density matrix evolution is given by

∂tρ=−i[H,ρ]+κ(2aρa†−{a†a,ρ}),

where the Hamiltonian is

H=δa†a+χa†a†aa+iε(a†−a).

The jump operator can be expressed as

L=2κa,

so that the dissipative part is in the standard Lindblad form.

2. Matrix Element of the Liouvillian Superoperator

In the coherent-state representation, we work with forward (+) and backward (−) contours. Denoting the fields by a+ and a− (with their complex conjugates a+* and a−*), we define the matrix element of the Liouvillian superoperator as

Lsuper(a+*,a+,a−*,a−)=−i[H(a+*,a+)−H(a−*,a−)]+κ[2a+a−*−a+*a+−a−*a−].

Here, the Hamiltonian evaluated on each contour is

H(a±*,a±)=δa±*a±+χa±*2a±2+iε(a±*−a±).
